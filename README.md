# Simple-Timer-Alarm

シンプルで使いやすい、Google Chrome 拡張機能のタイマー＆アラームです。
サイドパネルから手軽に操作でき、複数のタイマーやアラームを同時に管理できます。

## 主な機能

- **マルチタイマー**: 複数のタイマーを作成・同時実行できます。
- **マルチアラーム**: 特定の時刻に通知するアラームを複数設定できます。
- **サイドパネル UI**: ブラウザの作業を邪魔せず、横に表示して操作できます。
- **ドラッグ＆ドロップ**: タイマーやアラームの並び順を自由に入れ替えられます。
- **通知機能**: タイマー終了やアラーム時刻になると、ポップアップと音でお知らせします。
- **音量設定**: 通知音のボリュームをスライダーで調整可能です。
- **名称編集**: 各タイマーやアラームに好きな名前を付けられます。

## インストール方法

1.  このリポジトリをダウンロードまたはクローンします。
2.  Google Chrome を開き、アドレスバーに `chrome://extensions/` を入力して移動します。
3.  右上の「デベロッパー モード」をオンにします。
4.  「パッケージ化されていない拡張機能を読み込む」ボタンをクリックし、本プロジェクトのフォルダを選択します。

## 使い方

### 基本操作
- ツールバーの拡張機能アイコンをクリックすると、サイドパネルが開きます。
- 下部の「タイマー」/「アラーム」ボタンでモードを切り替えます。
- 「＋」ボタンで新しい項目を追加します。

### タイマー
- **開始/一時停止**: 再生/一時停止アイコンをクリックします。
- **リセット**: リセットアイコンをクリックすると元の時間に戻ります。
- **時間変更**: 時間表示部分（例: `03:00`）をクリックして直接編集できます（`分:秒` または `時:分:秒`）。
- **名称変更**: タイマー名をクリックして編集できます。
- **削除**: 右上の「×」ボタンで削除します。

### アラーム
- **ON/OFF**: スイッチを切り替えてアラームの有効/無効を切り替えます。
- **時刻変更**: 時刻表示部分（例: `08:00`）をクリックして編集できます（`時:分`）。
- **名称変更**: アラーム名をクリックして編集できます。

### 並び替え
- 各カードをドラッグして、好きな順番に並び替えることができます。

### 通知
- 時間になると通知ポップアップが表示され、音が鳴ります。
- **確認**: 音を止め、サイドパネルの終了表示を消します。アラームの設定は維持されます。
- **停止**: 音を止め、終了したタイマーをリセットしたりアラームをOFFにしたりします。

## ファイル構成

- `manifest.json`: 拡張機能の設定ファイル。
- `background.js`: バックグラウンドで動作するロジック（タイマー監視・音の制御等）。
- `sidepanel.html/js/css`: サイドパネルのUIと制御。
- `popup.html/js/css`: 通知用ポップアップのUIと制御。
- `offscreen.html/js`: 音声を再生するためのオフスクリーンドキュメント。
- `images/`: アイコンや各種ボタン画像。
- `sounds/`: 通知用音声ファイル。
